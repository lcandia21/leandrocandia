# Portafolio Web - Leandro Candia

## Descripción

Este proyecto consiste en un portafolio web personal desarrollado con HTML. Su objetivo es presentar información profesional, proyectos realizados y medios de contacto de forma clara y organizada.

## Estructura del proyecto

```text
proyecto/
│
├── index.html
├── README.md
├── assets/
│   └── Colopiogenes.jpg
│
└── pages/
    ├── inicio.html
    ├── sobremi.html
    ├── proyectos.html
    └── contacto.html
```

## Páginas

### Inicio

Presentación principal del portafolio con una breve descripción profesional e imagen destacada.

### Sobre mí

Información personal y profesional sobre el autor del sitio.

### Proyectos

Sección destinada a mostrar proyectos y trabajos realizados.

### Contacto

Información de contacto y medios para comunicarse con el autor.

## Tecnologías utilizadas

* HTML5
* Estructura semántica
* Navegación mediante enlaces internos
