# Portafolio Web - Leandro Candia

## Descripción

Este proyecto consiste en un portafolio web personal desarrollado con **HTML5** y **CSS3**. Su objetivo es presentar información profesional, proyectos realizados y medios de contacto mediante una interfaz clara, organizada y con un diseño visual atractivo.

## Estructura del proyecto

```text
proyecto/
│
├── index.html
├── README.md
├── styles.css
├── assets/
│   └── Colopiogenes.jpg
│
└── pages/
    ├── inicio.html
    ├── sobre-mi.html
    ├── proyectos.html
    └── contacto.html
```

> **Nota:** Si tu archivo CSS tiene otro nombre o se encuentra en otra carpeta, reemplazá `styles.css` por la ruta correspondiente.

## Secciones del sitio

### Inicio

Presentación principal del portafolio con una breve descripción profesional e imagen destacada.

### Sobre mí

Información personal y profesional del autor.

### Proyectos

Sección destinada a mostrar los proyectos y trabajos realizados.

### Contacto

Espacio con los medios de contacto para facilitar la comunicación con el autor.

## Tecnologías utilizadas

* HTML5
* CSS3
* Estructura semántica
* Navegación mediante enlaces internos

## Características

* Diseño con estilos personalizados mediante CSS.
* Estructura semántica utilizando etiquetas de HTML5.
* Navegación entre múltiples páginas.
* Organización de archivos mediante carpetas.
* Sitio de fácil lectura y mantenimiento.

## Cómo visualizar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador web moderno.
